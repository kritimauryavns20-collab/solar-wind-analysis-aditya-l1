# Solar Wind Particle Flux Analysis using Aditya-L1

This project analyzes solar wind particle flux data obtained from the Aditya-L1 mission.

The dataset used in this analysis comes from the ASPEX instrument, which measures solar wind particles and their energy distribution. The data were provided in Common Data Format (CDF) files and processed using Google Colab.

The objective of this project is to visualize and study variations in solar wind particle flux over time and identify patterns such as spikes or fluctuations in different energy channels.

The analysis includes:

* Reading CDF data files
* Plotting particle flux vs time
* Interpreting variations in solar wind behavior

Tools used:

* Python
* Google Colab
* Matplotlib
* cdflib
